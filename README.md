# generador_de_cartas
En este proyecto relizaremos un generador dinamico de cartas aleatorias basado en eventos del ciclo de vida del navegador. Al activarse el evento.onLoad, el motor ejecuta una logica de seleccion aleatoria entre los 4 palos de cartas que existen y la numeracion (2-9) incluyendo las letras (A, J, K).
